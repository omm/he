## Keyboard shortcuts

#### General

| Press this key | or this key    | To do this               |
|:---------------|:---------------|:-------------------------|
| F1             | Ctrl-Backslash | Show Command Palette     |
| Ctrl+P         |                | Quick Open, Go to File...|

#### Basic editing

| Press this key | or this key    | To do this              |
|:---------------|:---------------|:------------------------|
| Ctrl+X         |                | Cut line                |
| Ctrl+C         |                | Copy line               |
| Ctrl+V         |                | Paste                   |
